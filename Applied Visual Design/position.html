这里介绍position的一些相关问题

position-relative

当元素的定位设置为 relative 时，它允许你通过 CSS 指定该元素在当前文档流页面下的相对偏移量。

CSS 里控制各个方向偏移量的属性是 left、right、top 和 bottom。

它们代表从原来位置向远离该方向偏移指定的像素、百分比或者 em。

absolute and fixed

fixed 和 absolute 的最明显的区别在于，fixed定位的元素不会随着屏幕滚动而移动。
